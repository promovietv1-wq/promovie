# PromoVie

Site institucional estático da PromoVie.

## Arquivos

- `index.html`: site completo atual.
- `netlify.toml`: configuração para publicar na Netlify.
- `_redirects`: fallback para manter navegação funcionando.
- `.nojekyll`: evita processamento automático no GitHub Pages.

## Netlify

- Build command: deixe vazio
- Publish directory: `.`

Basta subir todos estes arquivos no repositório do GitHub e conectar esse repositório na Netlify.
